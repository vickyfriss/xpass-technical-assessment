# xPass Technical Assessment - Player Passing Performance in Football

This repository contains a **technical assessment project** completed as part of a data science evaluation.  
It demonstrates the development, calibration, and application of an **Expected Passes (xP) model** to analyse player passing performance, identify risk-taking and conservative passers, and generate actionable football insights.  

The project workflow includes:

- Data extraction and preprocessing from StatsBomb Free Data
- Feature engineering for probabilistic modelling of pass completion
- Model fitting, calibration, and validation
- Player-level analysis to highlight extreme passers and tactical patterns
- Visualisation of passing profiles for decision support

> The analysis is designed to be interpretable, reproducible, and applicable to real-world football decision-making, aligning with typical technical assessment expectations for a professional football club data scientist role.
